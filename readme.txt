== Description ==
[kuzi] is Joke Plugin.
By adding a tag to the template, roulette appears.
"Yes or No" will appear when you press the button on the roulette.

== Installation ==

1. Upload all the php files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php mtdKuzi(); ?>` in your templates
